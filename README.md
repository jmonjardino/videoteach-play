## Overview

Viora is a modern, AI-powered learning platform where instructors can create video-based courses and students can enroll for free. Teachers can optionally set a price that learners see for context, but enrollment remains free to keep education accessible.

## Getting Started

Prerequisites:

- Node.js and npm installed (recommended via `nvm`)

Steps:

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm i
npm run dev
```

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase (Auth, Database, Storage)

## Deployment

You can deploy Viora using your preferred hosting provider or static site hosting for the frontend and Supabase for backend services.
